# INSTALL FOR LINUX

`git clone https://github.com/sonvan1811/getproxy.git`

`cd getproxy`

`python3 -m pip install -r requirements.txt` or `pip3 install -r requirements.txt`

`python3 gp.pt`

# INSTALL FOR WINDOWS

Download source code file and UNZIP File

`cd getproxy-master`

`python gp.py`
